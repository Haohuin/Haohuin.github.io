##数据类型：

(1) Java是一种强类型语言，针对每种数据都提供了对应的数据类型。
(2) 分类：
A. 基本数据类型
B. 引用数据类型：类，接口，数组。
(3) 类型转换
i. boolean类型不参和其他类型进行转换。
ii. 默认转换：要从小到大
A. byte,short,char --> int --> long --> float --> double
B. byte,short,char之间不相互转换，直接转成int类型参与运算。即使是同一个数据类型的运算，也会先转换成int类型。
(4) 运算
i. 字符运算相当于ASCII码int型运算
ii. 字符串运算相当于拼接

##变量：
1.成员变量（instance variable）：又叫做对象变量、类成员变量
随着对象的建立而存在堆内存中。
2.局部变量（local）：声明在方法中的变量。在栈中
只存在于方法中，无法在方法外进行任何操作，
3.类变量（静态变量）static+变量
是一种比较特殊的实例变量，是类中独立于方法之外的变量，用static关键字修饰。在程序加载时就为它在堆中开辟了内存，而且静态变量的生命周期将一直持续到整个系统关闭。

##运算符：
(1) 算术运算符
i. ++和–放在操作数后面的原理 举例：a = a++;
int temp = a;
a++;
a = temp;
(2) 赋值运算符
i. 赋值运算符“=”特点
A. 赋值运算符从右向左结合
B. 赋值表达式有值=右边表达式的值
ii. 扩展运算符“+=”，“-=”等等
A. 隐含了自动强制转换
(3) 比较运算符
i. 最终结果始终是boolean类型
(4) 逻辑运算符
i. ^:相同则false,不同则true
ii. &&和&的区别，&&有短路效果
(5) 位运算符
a: 判断一个数是否是奇数 a & 1 == 1;
b: 对2 ^ n 取余。 a & (2^n - 1);
c: 如何判断一个数是否是2^n a & (a-1) == 0
d: 两个整数之间的交换；
e: 有一堆数，都是成对出现的，只有一个单独的，找出这个单独的数
f: 用最有效率的方式求2 * 8的值 2 << 3;
